# Freddle

This a fork of [react-wordle](https://github.com/cwackerfuss/react-wordle), modified so it's just about my cat Fred:

![Fred](./public/logo192.png)

[**Try it out!**](https://freddle.com/)

## Build and run

### To Run Locally:

```bash
npm install
npm run start
```

### To Deploy:

```
npm run deploy # deploys to S3
```
